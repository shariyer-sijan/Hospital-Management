# CareSync Hospital Management

## Project Title
CareSync: Integrated Patient & Staff Management Core

## Description
CareSync is a simple hospital management system built using HTML, CSS, and JavaScript.  
This project helps manage hospital-related activities like patient serial booking, doctor availability checking, login system, dark mode, and data handling using localStorage.

---

## Features

- Login & Logout System
- Create Account
- Doctor Availability Checker
- Patient Serial Booking
- Dark/Light Mode
- Responsive Design
- API Integration
- Local Storage Data Handling
- Form Validation
- Dynamic JavaScript Interactions

---

## Technologies Used

- HTML
- CSS
- JavaScript
